# Gogolook Web Design System

A comprehensive web design system documenting Gogolook's UI components, design tokens, and patterns for building brand-compliant web interfaces.

## What's Inside

- **Design Tokens** — Colors, typography, spacing, shadows, border radius, gradients
- **Components** — Buttons, cards, inputs, selection controls, badges, navigation, tabs, alerts, avatars, dividers, progress, tooltips
- **Patterns** — Layout grid, dark mode, animation, accessibility guidelines
- **Live Previews** — All components rendered as interactive HTML demos

## Quick Start

Open `index.html` in a browser. Password: `gogolook2026`
