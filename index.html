Gogolook Web Design System
A comprehensive component library and design token system for building Gogolook-branded web interfaces. Based on gogolook.com patterns, optimized for consistency, accessibility, and developer ergonomics.

This design system provides copy-paste ready CSS tokens, live component previews, and usage guidelines. All components use the gl- prefix for CSS custom properties and follow WCAG AA accessibility standards.

Buttons

Four variants: Primary (main CTA), Secondary (outline), Ghost (text-only), and Dark (on light backgrounds). Three sizes: small, default, and large.

Sizes
States
With Icon
DO
Use Primary for the single most important action on the page. Use Secondary for alternative actions alongside Primary.
DON'T
Don't use multiple Primary buttons in the same section. Don't use Ghost buttons for important actions.
Cards

DO
Only animate transform and opacity. Respect prefers-reduced-motion. Keep hover transitions under 200ms.
DON'T
Never use bounce, elastic, or spring easings. Don't animate layout properties (width, height, top, left).
Accessibility

All components meet WCAG AA standards. Minimum 4.5:1 contrast ratio for normal text, 3:1 for large text and UI elements.

Contrast Pairs
PairRatioStatus
Black on White21:1Pass
White on Gogolook Blue4.6:1Pass
White on Shadow Blue13.5:1Pass
White on Clear Horizon3.1:1Large only
White on Neo Green2.8:1Fail
Requirements Checklist
• All interactive elements reachable via Tab
• Focus visible: outline: 2px solid var(--gl-blue); outline-offset: 2px
• Escape key closes modals and dropdowns
• Skip-to-content link as first focusable element
• Images have alt text; decorative images use alt=""
• Form inputs have associated <label> elements
• Buttons use <button>, links use <a>
• Heading hierarchy (h1 → h2 → h3) without skipping levels
• prefers-reduced-motion disables all animations
DO
Use Neo Green and Clear Horizon only for decorative elements, badge backgrounds, or large bold text on white backgrounds.
DON'T
Never use Neo Green or Clear Horizon as body text color on white. Never rely on color alone to convey information.
