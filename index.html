Buttons

All buttons are pill-shaped (border-radius: 100px). Four variants: Primary (filled blue), Outline, White, and Ghost. Most CTAs include an arrow icon (→).

Variants
Sizes
States
DO
Always include an arrow (→) in CTA buttons. Use Outline style for dual CTAs side-by-side ("For Personal" / "For Business").
DON'T
Never use square or small-radius buttons. Never use more than one Primary filled button in the same section.
